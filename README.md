# Amazon IoT Events (amazon-iot-events)
AWS IoT Events is a fully managed IoT service that makes it easy to detect and respond to events from IoT sensors and applications. It enables you to monitor your equipment or device fleets for failures or changes in operation, and trigger actions when such events occur.

## APIs

### AWS IoT Events API
The AWS IoT Events API provides programmatic access to create and manage detector models, input definitions, and alarms for detecting and responding to events from IoT sensors and applications.
